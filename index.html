Introduction

CityPlan CSS is essentially a remake of Angular Material's CSS layout with some ideas taken from other popular CSS grid systems like Pure or Tachions.

It is mobile-first, responsive, and based on CSS3 flexbox.

Getting Started

Copy the line of code below and paste it in the head of the html file(s) you want to include tachyons in. 
<link rel="stylesheet" href="https://unpkg.com/cityplan@latest/dist/cityplan.min.css">
or install via npm: 
npm install cityplan --save

Container Layouts

CityPlan CSS consist of two types of classes: the container classes (cp-c-*) and box classes (cp-b-*).

Use the .cp-c-row and .cp-c-column classes on a container element to specify the layout direction for its children: horizontally in a row or vertically in a column.

Layouts and Responsive Breakpoints

To make your layout automatically change depending upon the device screen size, use one of the following classes:

Key CSS Media Query Applies Classname
None None Always .cp-c-*.cp-b-*
sm @media screen and (min-width: 35.5em) 568px .cp-c-sm-*.cp-b-sm-*
md @media screen and (min-width: 48em) ≥ 768px .cp-c-md-*.cp-b-md-*
lg @media screen and (min-width: 64em) ≥ 1024px .cp-c-lg-*.cp-b-lg-*
xl @media screen and (min-width: 80em) ≥ 1280px .cp-c-xl-*.cp-b-xl-*

Children within a Layout Container

To customize the size and position of elements in a layout container, use the .flex, .order, and .offset classes on the container's child elements:

Add the .flex class to a layout's child element and the element will flex (grow or shrink) to fit the area unused by other elements. Flex defines how the element will adjust its size with respect to its parent container and the other elements within the container.

Additional Flex Values

Classname Description
.flex Will grow and shrink as needed. Starts with a size of 0%. Same as .flex-0.
.flex-none Will not grow or shrink. Sized based on its width and height values.
.flex-initial Will shrink as needed. Starts with a size based on its width and height values.
.flex-auto Will grow and shrink as needed. Starts with a size based on its width and height values.
.flex-grow Will grow and shrink as needed. Starts with a size of 100%. Same as .flex-100.
.flex-nogrow Will shrink as needed, but won't grow. Starts with a size based on its width and height values.
.flex-noshrink Will grow as needed, but won't shrink. Starts with a size based on its width and height values.
