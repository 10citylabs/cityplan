Getting Started

Copy the line of code below and paste it in the head of the html file(s) you want to include tachyons in. 
<link rel="stylesheet" href="https://unpkg.com/cityplan@latest/dist/cityplan.min.css">
or install via npm and import it. 
npm install cityplan --save

Responsiveness

CityPlan is mobile-first and responsive, and you're able to customize the grid by specifying CSS Media Query breakpoints and grid classnames.
Key Applies Classname
None Always .row
@media screen and (min-width: 35.5em) ≥ 568px .md-row
@media screen and (min-width: 48em) ≥ 768px .lg-row
